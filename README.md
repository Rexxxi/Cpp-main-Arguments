# C-Arguments
How to use C++ main Arguments with console 

This code example shows you how to use arguments for your
main method.

How to enter args with console:
Open cmd -> choose the exe. with command cd, and write the arguments begind it
like that:   C:\Users\Test\Documents\test.exe arg1 arg2 arg3 ...

arg1, arg2, ... will appear in the string array in the code.
